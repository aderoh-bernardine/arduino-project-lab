# Arduino Project Lab

This repository contains basic Arduino projects I am building as I learn embedded systems and electronics.

## 📁 Projects

### 1. LED Blink
A simple project to turn an LED ON and OFF repeatedly.

### 2. 4-Bit Binary Counter (1–15)
Displays numbers from 1 to 15 in binary using LEDs.

### 3. Dimmable LED Control System
Controls LED brightness using a potentiometer and PWM.

### 4. Traffic Light Control System
Simulates a traffic light sequence using red, yellow, and green LEDs with timed delays.

## 🚀 Goal
To build a strong foundation in Arduino programming and hardware interfacing through hands-on projects.
